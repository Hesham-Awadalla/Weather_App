API-Daten für die Visualisierung im Frontend
===================================================================================================

location:	- name			--> Name der Stadt
			- country		--> Land
			- localtime		--> Letzter Zeitpunkt an dem die Daten geladen wurden


current:	- temp_c		--> Temperatur in Celsius
			- feelslike_c	--> Gefühlte Temeratur 
			- wind_kph		--> Windgeschwindigkeit

			- condition
				- text		--> Beschreibung Wetter
				- icon		--> Icon zur Visualisierung

===================================================================================================
Funktionen im Frontend
===================================================================================================

- Beim Start der Software werden drei Default Städte geladen (ist das anpassbar?) und der aktuelle Standort

- Button 'LocateMe'			--> Holt sich den aktuellen Standort des Nutzers und liefert ihm die Wetterdaten

- TextBox 'SearchCity'		--> Beim Eingeben von einzelnen Zeichen wird aus der Datenbank eine gefilterte Liste angezeigt
Autocomplete Textbox			Wird eine spezifische Stadt gewählt wird 'LocateCity' aufgerufen welche die Location-Werte liefert
								und anschließend im Frontend darstellt

- Button 'Refresh'			--> Der Nutzer kann den Refresh-Button betätigen, welcher aktuelle Datenlädt

(x) Im Hintergrund soll alle 3 Minuten ein erneuter Refresh durchgeführt werden
